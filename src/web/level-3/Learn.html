李益敏副局长参加省军供中心党支部专题学习讨论会

信息来源：浙江省粮食和物资储备局 发布时间：2022-05-07 11:02 作者：省军供中心 浏览次数：37

5月6日下午，省军供中心党支部组织学习潘建漳书记在5月5日全局干部职工大会上的讲话精神，李益敏副局长参加了会议并结合军供工作提出具体要求。

会上，中心全体党员干部职工结合实际进行了讨论交流，大家纷纷表示，潘书记的讲话主题鲜明、重点突出、求真务实，接地气、出实招、振精神，有很强的针对性、指导性和感染力、渗透力。 作为军供人，要把思想和行动统一到会议精神上来，克服“躺平惰性”思想，秉持“在职一天做好一天”的理念，营造“有为才有位”的工作作风，树立团结友爱、奋进向上的良好氛围，以钉钉子精神抓好军粮供应服务各项工作。

李益敏副局长对大家的发言表示肯定，他强调，潘书记在讲话中指出了存在的问题，代表局党组亮出了观点和态度，下一步省军供中心要一以贯之讲政治，对军供工作的认识要再提高、对国家和省里的要求要再贯彻、对局党组的决策要严落实； 要一以贯之讲正气，坚决抵制各种歪风邪气；要一以贯之讲团结，树立风清气正的政治氛围。全体军供人要结合实际进一步深化学习，统一思想认识，聚焦中心工作，在局党组的领导下，凝心聚力、团结一心，朝着共同的目标努力再努力，以优异成绩迎接党的二十大和省第十五次党代会胜利召开。
